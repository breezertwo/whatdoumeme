# whatdoumeme

This is a simple server & client for playing a version of ["What Do You Meme®"](https://whatdoyoumeme.com/) with your own custom cards and memes. Created to play with friends.

Info: The game was created in 2021. In 2026 i updated the broken dependencies and removed webpack. It lacks many security features and is not suitable for production environments, only use for fun or in controlled environments.
